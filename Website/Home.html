What is this about?


temperature

Visualize the Summer Annual Composites layers displaying the LST (°C) median values for Salzburg.


hexagonal bins

Click on the Hexagons to enable a pop-up showing the spatially aggregated LST (°C) values by mean.

Summer period selection

Use the Summer Period dropdown list to select the year of interest from 2018 to 2022.



swipe

Compare the Annual Composites layers with the spatial aggregated hexagons by using the swipe.



locate

Look for a place in Salzburg with the search button or using the bookmarks.
